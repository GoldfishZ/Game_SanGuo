游 戏 规 则

以下说明以当前 Web 与训练环境共用的权威战斗规则为准。卡面上的圆形字章代表武将特性；战斗中出现的“待判”“已判”标签会提示特性状态。

一、选将、布阵与胜负

选将

双方分别从自己的候选池选将，所选武将费用合计不能超过 8。至少选择一名武将；不同稀有度和费用不直接改变规则，只影响武将本身数值与技能。

阵型

权威阵型为 3层×4路：每一路由前卫、中坚、后卫依次保护。画面为了让双方相向而战，会显示成4排×3列；越靠近战场中央的卡越靠前。

普攻目标

通常只能选择每一路最靠前且可被选中的敌人。前卫阵亡后，身后的武将才会暴露。伏兵、强制攻击和“只能攻击正前方”等状态会进一步改变合法目标。

胜负与超时

一方所有武将最终阵亡即落败；复活成功的武将仍算存活。达到战斗回合上限仍未分胜负时，统一判定为平局。

二、回合、士气与伤害

回合行动

掷骰决定先后手。轮到一方时，可让存活武将发动尚可用的主动技能和普通攻击；每名武将通常每回合至多使用一次技能、一次普攻，攻速判定成功时可获得一次追加普攻。

独立士气池

双方各自拥有士气，并非共用。初始士气与上限均为12；后手获得补偿，上限和当前士气各增加2。主动技能按卡面标注消耗士气，结束己方行动时恢复2点，但不会超过上限。

普攻伤害

若攻击方武力高于目标，伤害为双方武力差；否则比较双方“武力+智力”，差值最多按3点计算。最终基础普攻伤害最低为1，再依次经过勇猛、防栅、护盾、连计等规则修正。

奇偶判定

需要判定时，玩家先押“奇”或“偶”，系统再掷一枚六面骰。骰点奇偶与选择一致即成功。攻速、勇猛和魅力是彼此独立的判定，不会共用同一次押注。

三、七项武将特性

勇勇猛低血反攻

触发条件：发动普攻时，自身当前生命必须严格低于最大生命的一半，恰好一半不会触发。

判定效果：攻击方猜奇偶；成功则本次普攻伤害×1.5并四舍五入，失败则保持原伤害。卡面会从“未蓄势”变为“待判”，结算后显示成功或失败。

魅魅力绝命反噬

触发条件：武将受到致命伤害，并且在复活等效果结算后仍然阵亡。没有伤害来源时不会反噬。

判定效果：被攻击方猜奇偶；成功则对伤害来源造成“本次致命伤害的一半”，四舍五入。魅力可能反杀攻击者，失败则不造成伤害。

募募兵持续恢复

触发条件：该武将存活，且在己方新回合开始时生命未满。

效果：自动回复1点生命，不超过生命上限。若已满血、已经阵亡，或尚未轮到己方新回合，则不会恢复。

栅防栅一次格挡

触发条件：第一次受到普通攻击。技能伤害不会消耗防栅；拥有“无视防栅”效果的普攻可以绕过它。

效果：将该次普攻伤害完全降为0并使防栅破碎。当前规则中，破碎后的防栅本局不会自动重建。

连连计同荣共损

共享状态：己方多名存活的连计武将会同步所持增益和减益；其中一人获得或失去相关状态时，连计体系会重新同步。

分担伤害：连计武将受伤时，伤害按存活连计人数整除，并让每名连计武将承受分担后的数值。阵亡者不再参与后续共享。

生复活每局一次

触发条件：武将首次因伤害降至0生命。

效果：立即恢复为最大生命的一半（向下取整）并继续留在战场，每局仅能成功一次。因为武将没有最终阵亡，同一次伤害不会再触发魅力。

伏伏兵隐匿反击

隐藏：隐藏期间不能被敌方普通攻击直接选中；使用自己的主动技能、完成伏击反击，或队友全部阵亡后会破隐。

反击：相邻8格内的友军遭到普攻时，一名尚未触发的隐藏伏兵会对攻击者造成原伤害一半、最低1点的反击伤害。每名伏兵每局只能反击一次，且不会交换阵位。
