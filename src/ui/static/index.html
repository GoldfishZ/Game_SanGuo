游 戏 规 则

阵型

3行x4列方格，前列武将挡住后列。敌方只能攻击最前排武将。

伤害

攻击方武力>目标武力: 伤害=武力差。攻击方武力<=目标武力: 伤害=min(3,(武力+智力)差)，最低1点。

士气

初始12点，共用士气池。技能消耗士气。后手士气上限+2。

回合

先手->后手交替。每回合: 1技能阶段->2普攻阶段。

防栅

完全抵挡一次攻击后失效。被攻击时蓝色护盾守卫，抵挡后护盾碎裂消失。

王异 · 防栅

连环

所有连环武将共享buff/debuff并分担伤害。金色锁链相连，一荣俱荣一损俱损。

诸葛亮 · 连环

勇猛

低血量时普攻伤害x1.5。生命值降至一半以下时触发血红色战意特效。

张飞 · 勇猛

魅力

被击杀时反弹致死伤害的一半给攻击者。死亡时紫色魅影反噬仇敌。

蔡文姬 · 魅力

募兵/复活/伏兵

募兵: 受伤后每回合回复1点。复活: 死亡时50%HP复活一次。伏兵: 使用技能前不可被选中。

许褚·募兵
孙坚·复活
张任·伏兵

胜利

一方全体武将阵亡则游戏结束。
